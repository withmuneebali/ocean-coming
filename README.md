# The Ocean Within Travel

Welcome to the official repository for **The Ocean Within Travel**, a luxury travel agency dedicated to curating bespoke, high-end journeys for discerning clients.

## Overview
The Ocean Within Travel specializes in handcrafted, unforgettable travel experiences. This repository contains the source code for our company website, built with modern web technologies to provide a seamless and elegant user experience.

## Live Demo
Experience our vision at: [https://oceanwithtravel.com](https://oceanwithtravel.com)

## Tech Stack
Our website is built using a modern, performant stack:
* **Framework:** [React](https://react.dev/)
* **Build Tool:** [Vite](https://vite.dev/)
* **Language:** TypeScript
* **Styling:** Tailwind CSS
* **Deployment:** GitHub Pages (via GitHub Actions)

## Project Structure
- `/src`: Contains the core React components and business logic.
- `/public`: Static assets, logos, and images.
- `.github/workflows`: Automated CI/CD pipeline configuration.

## Deployment
This project is automatically deployed to GitHub Pages upon every push to the `main` branch using GitHub Actions.
